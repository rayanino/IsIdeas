# capture/

This directory is for raw idea intake.

Use it for sparks that are worth preserving but not yet trusted enough to enter the serious registry.

## What Belongs Here
- fast captures
- rough sketches
- early objections
- early leverage guesses

## What Does Not Belong Here
- mature dossiers
- formal specs
- serious registry entries

## Rule
The purpose of `capture/` is preservation without premature promotion.
