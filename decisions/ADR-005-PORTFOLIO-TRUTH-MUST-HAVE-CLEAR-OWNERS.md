# ADR-005 — Portfolio truth must have clear owners

## Status
Accepted

## Decision
The repository will explicitly define which artifacts are authoritative for which kinds of state.

## Why
Without clear ownership of truth, the repo will accumulate conflicting summaries, stale status labels, and silent drift between dossier reasoning and portfolio control.

## Consequence
The registry, workspace README files, dossiers, focus artifact, and priority board now have distinct roles rather than overlapping authority.
