# Portfolio Prioritization Framework

The portfolio now uses four live bands.

## Frontier

Ideas receiving real pressure, active challenge, and dossier-hardening attention.

## Challenge

Serious candidates worth keeping alive, but not yet trusted enough for frontier attention.

## Watch

Interesting directions that should stay visible without consuming serious hardening time.

## Parked

Directions intentionally set aside.

## Main Rule

Bandwidth should stay narrow at the top.
The existence of a directory does not entitle an idea to frontier status.
